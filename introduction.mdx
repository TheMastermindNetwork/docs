# Mastermind Playbook

AI UGC (User Generated Content) means creating videos that look like real people genuinely talking about a product — built entirely with AI tools. No camera, no actors, no studio.

The full pipeline in order:

```
Script → Character → Voice → Lip Sync → Edit
```

Every video you make follows this exact sequence. Learn each step once and you can produce 30+ videos in a day.
