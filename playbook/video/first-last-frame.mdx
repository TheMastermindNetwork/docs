# First Frame + Last Frame Method

<Info>
  Source: [https://x.com/recap_david/status/2026360278645092862](https://x.com/recap_david/status/2026360278645092862)
</Info>

Instead of prompting from scratch, you provide both a start frame and an end frame. The model fills in the motion between them.

This gives you **complete control over composition** while letting AI handle the cinematic movement.

## How to use it

1. Generate your start frame in Nano Banana (character in position A)
2. Generate your end frame in Nano Banana (character in position B)
3. Upload both to Kling 3.0 as first and last frame
4. Add your motion prompt
5. Generate

## Prompt elements for premium fashion feel

```
"slow cinematic push-in"   // camera slowly moves toward subject
"shallow depth of field"   // background blurs naturally
"micro-parallax"           // subtle layered depth movement
```

The output looks like a high-budget fashion shoot. Cost: the price of a Kling generation.
