# LTX 2.3 — Audio + Image Lip Sync in ComfyUI

<Info>
  Source: [https://x.com/AIWarper/status/2029976759236710817](https://x.com/AIWarper/status/2029976759236710817)
</Info>

LTX 2.3 runs in ComfyUI and accepts an audio clip plus a character image to generate a talking head video.

- **Output:** 10 seconds at 720p, 24fps
- **Speed:** under 25 seconds render time
- **Cost:** free if running locally

## When to use this

If you're already using ComfyUI for other workflows, this integrates cleanly. The speed is fast enough for rapid iteration — generate, review, tweak, regenerate without waiting.

## Setup

Install the LTX 2.3 node in ComfyUI via the node manager. Connect an image input, an audio input, and the output to a video preview node.
