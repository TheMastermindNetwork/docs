# Vision2JSON — Steal Any Image's Style

<Info>
  Source: [https://x.com/ViralOps_/status/2029556354692874531](https://x.com/ViralOps_/status/2029556354692874531)
</Info>

You find a photo you love on Pinterest. You run it through this Gemini Gem. It gives you a complete JSON describing every visual detail — lighting, skin texture, camera angle, color palette, everything. You paste that JSON into Nano Banana to recreate the same style with your own character.

## Setup (one time)

1. Open Gemini → click **Gems** → **New Gem**
2. Name: `Vision2JSON`
3. Paste the full system prompt from the tweet link above
4. Save

## How to use

1. Find a reference image on Pinterest
2. Paste it into your Vision2JSON gem
3. Select **Pro model**
4. Hit enter — you get a detailed JSON back
5. Copy that JSON into Nano Banana Pro

## Best for

- Matching the exact iPhone selfie aesthetic
- Golden hour lighting
- Specific color palettes for different demographics
- Any vibe you've seen on Pinterest and want to replicate
