# Before/After Character — Transformation Ads

<Info>
  Source: [https://x.com/maxxmalist/status/2027420756200607864](https://x.com/maxxmalist/status/2027420756200607864)
</Info>

For transformation ads (weight loss, skincare, fitness), you need two versions of the same character — before and after. Here's how to do it while keeping the face identical.

## Step 1 — Create your base character

Write a detailed Nano Banana prompt covering age, skin tone, freckles, hair, outfit, setting, and lighting. Be specific about everything.

Example:
```
24-year-old white woman, warm fair complexion, light freckles, warm brown eyes,
honey blonde wavy hair half-up half-down, oversized lavender crewneck, sitting
on tan velvet couch, cozy bedroom, warm fairy lights, realistic iPhone front
camera, no filter.
```

## Step 2 — Create the "before" version

Attach the original image as reference, then add:

```
Maintain exact same identity (same freckles, eyes, eyebrows, lips, hair, clothing).
Only change: face appears visibly bloated and puffy, cheeks rounder and swollen,
jawline soft and undefined, slight under-eye puffiness, skin looks duller and tired.
```

This keeps the same person but shows the before state — critical for believable transformation content.

## Step 3 — Animate with Kling

Upload each still image as the starting frame. Kling animates it with the motion you prompt. Keep clips to 3–5 seconds each — assemble in CapCut.
